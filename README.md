## Requirements

- iOS 12.1+
- Xcode 12.4+
- Swift 5.0+

## Installation

MSRichLinkPreview is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MSRichLinkPreview'
```
